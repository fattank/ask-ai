# ASK-AI
Project Name: ASK-AI  A logical framework for assessing personal health concerns and navigating medical care decisions. This is not a diagnostic tool, but a structured thinking model to help you clearly articulate symptoms, rationally triage their urgency, and determine the safest next steps.

# AI 医疗提问指南 / AI Medical Question Guides

![DeepSeek Logo](./deepseek-logo.png)

> 教你如何更安全、更清晰地向 AI 提健康相关问题。  
> 目标：**帮助沟通，不替代医生。**

## 1. 项目简介

本仓库收集了两套**向 AI 提医疗问题的实用模版**：

- 面向**普通人 / 患者**：  
  帮你把自己的不舒服说清楚，  
  更好地和 AI 沟通、准备去看医生。

- 面向**基层医生 / 临床医生**：  
  帮你在遇到**急、重、怪、没把握**病人时，  
  用 AI 做一次“思路体检”：扩展思路、查漏补缺、减少误诊和延误救治。

## 3. 快速开始

### 3.1 普通人 / 患者怎么用

- 只需要记住一句话：

> **3 句话说清情况 + 3 个问题问 AI**

- 打开：[`PATIENT_GUIDE_zh.md`](./PATIENT_GUIDE_zh.md)，  
  里面有你可以**直接复制修改**的提问模版，例如：
  - 如何用 3 句话说清自己的情况  
  - 如何问「要不要去医院、去哪里看」  
  - 看医生前要准备什么、怎么跟医生说

### 3.2 基层医生怎么用

- 核心用法：

> **先说清自己的想法，再请 AI：  
> 想得更广、故意挑错、帮你排先后。**

- 打开：[`DOCTOR_GUIDE_zh.md`](./DOCTOR_GUIDE_zh.md)，  
  里面包括：
  - 初次使用时的 AI 角色设定文本  
  - 如何用几句话交代病例  
  - 三个关键“动作”：  
    - 让 AI 自己全面想一遍  
    - 让 AI 故意唱反调，指出盲区  
    - 让 AI 帮你按风险排优先级  
  - 以及：不建议使用 AI 的急危重场景、隐私合规提醒等。

## 4. 重要声明

- 本项目内容仅用于**健康科普和沟通辅助**。  
- **AI 不能替代线下医生的面对面诊疗和处方决策。**  
- 任何关于**诊断、用药、检查、手术、转诊**等医疗决策，  
  必须由具有合法资质的医生，在充分评估后做出判断。
